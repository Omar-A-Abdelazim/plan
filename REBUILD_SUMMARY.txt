BEFORE (Broken):
  ├─ ❌ Circular dependency: widget.js → other modules → widget.js
  ├─ ❌ Duplicate state: Timer in main app AND widget
  ├─ ❌ No real-time sync between windows
  ├─ ❌ Complex prop drilling
  └─ ❌ Non-functional widget settings

AFTER (Clean):
  ├─ ✓ Single Source of Truth: PomodoroContext is authoritative
  ├─ ✓ IPC-Based Sync: Widget subscribes to all state updates
  ├─ ✓ Real-Time Synchronization: Changes broadcast instantly
  ├─ ✓ Independent Widget: Can run standalone timer
  ├─ ✓ Full Task Management: Add, complete, delete tasks
  ├─ ✓ Language Support: EN/AR with proper quotes
  ├─ ✓ Clean Separation: Widget is vanilla HTML/JS, no React
  └─ ✓ Draggable Window: Header-only drag region

💾 STATE FLOW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PomodoroContext (Main App)
  ├─ timerState: { mode, timeRemaining, isRunning, sessionCount, totalSessions }
  ├─ settings: { workDuration, shortBreakDuration, longBreakDuration }
  ├─ currentQuote: string
  ├─ tasks: [{ id, text, completed }]
  └─ language: 'en' | 'ar'
         ↓ (useEffect triggers on any state change)
         ↓ window.electronAPI.updateWidgetState(state)
         ↓
    IPC: update-widget-state
         ↓ (main process broadcasts)
         ↓ widgetWindow.webContents.send('widget-state-update', state)
         ↓
    Widget Process (widget.html)
         └─ Receives state, renders UI immediately

Open Widget:
  • Tray menu → "Open Pomodoro Widget"
  • Hotkey: Ctrl+Shift+P
  • From code: window.electronAPI.openWidget()

Widget Controls:
  • Play/Pause   → Start/stop timer
  • Reset        → Back to start
  • Add Task     → Type + Enter or click Add
  • Complete     → Checkbox
  • Delete       → × button
  • Language     → EN/AR buttons
  • Close        → × in header
